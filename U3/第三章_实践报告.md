# 1 题目描述

用 Jacobi 迭代法和 Gauss-Seidel 迭代法求解方程组：

$$
\begin{bmatrix}
10 & -2 & -1 \\
-2 & 10 & -1 \\
-1 & -2 & 5
\end{bmatrix} \begin{bmatrix}
x_{1} \\
x_{2} \\
x_{3}
\end{bmatrix} = \begin{bmatrix}
3 \\
15 \\
10
\end{bmatrix}
$$

取初值 $\mathbf{x}^{(0)} = [0,\ 0,\ 0]^{T}$，要求满足 $\max_{1 \leq i \leq 3}|x_{i}^{(k+1)} - x_{i}^{(k)}| \leq 10^{-5}$ 时迭代终止，比较两种方法的迭代次数。

# 2 程序代码

此节设计了一个 C 语言程序代码，实现了 Jacobi 迭代法和 Gauss-Seidel 迭代法，并完成了精度要求下的计算结果和效率比较。程序代码如下：

```c
/*
 * 计算方法实践作业 - 第三章第三题
 * 使用Jacobi迭代法和Gauss-Seidel迭代法求解线性方程组
 * 方程组：
 *   10*x1 - 2*x2 -  x3 = 3
 *  -2*x1 + 10*x2 -  x3 = 15
 *  - x1 - 2*x2 + 5*x3 = 10
 * 初始值：x0 = [0, 0, 0]
 * 误差容限：1e-5
 * 比较两种方法的迭代次数
 */
#include <stdio.h>
#include <math.h>

#define N 3
#define MAX_ITER 1000
#define TOL 1e-5

/* 打印向量 */
void print_vector(double x[N], const char* name) {
    printf("%s = [", name);
    for (int i = 0; i < N; i++) {
        printf("%.10f", x[i]);
        if (i < N - 1) printf(", ");
    }
    printf("]\n");
}

/* Jacobi迭代法求解线性方程组
 * A: 系数矩阵, b: 右端向量, x: 初始猜测/输出解
 * 返回迭代次数 */
int jacobi(double A[N][N], double b[N], double x[N]) {
    double x_new[N];
    int iter = 0;
    
    while (iter < MAX_ITER) {
        for (int i = 0; i < N; i++) {
            double sum = 0.0;
            for (int j = 0; j < N; j++) {
                if (i != j) {
                    sum += A[i][j] * x[j];
                }
            }
            x_new[i] = (b[i] - sum) / A[i][i];
        }
        
        double max_diff = 0.0;
        for (int i = 0; i < N; i++) {
            double diff = fabs(x_new[i] - x[i]);
            if (diff > max_diff) max_diff = diff;
            x[i] = x_new[i];
        }
        
        iter++;
        if (max_diff < TOL) {
            break;
        }
    }
    
    return iter;
}

/* Gauss-Seidel迭代法求解线性方程组
 * A: 系数矩阵, b: 右端向量, x: 初始猜测/输出解
 * 返回迭代次数 */
int gauss_seidel(double A[N][N], double b[N], double x[N]) {
    int iter = 0;
    
    while (iter < MAX_ITER) {
        double max_diff = 0.0;
        for (int i = 0; i < N; i++) {
            double sum = 0.0;
            for (int j = 0; j < N; j++) {
                if (i != j) {
                    sum += A[i][j] * x[j];
                }
            }
            double x_new = (b[i] - sum) / A[i][i];
            double diff = fabs(x_new - x[i]);
            if (diff > max_diff) max_diff = diff;
            x[i] = x_new;
        }
        
        iter++;
        if (max_diff < TOL) {
            break;
        }
    }
    
    return iter;
}

/* 主函数：初始化并调用两种迭代方法 */
int main() {

    double A[N][N] = {
        {10.0, -2.0, -1.0},
        {-2.0, 10.0, -1.0},
        {-1.0, -2.0, 5.0}
    };
    double b[N] = {3.0, 15.0, 10.0};
    double x_jacobi[N] = {0.0, 0.0, 0.0};
    double x_gs[N] = {0.0, 0.0, 0.0};
    
    printf("迭代法解线性方程组:\n");
    printf("系数矩阵A = [[10, -2, -1], [-2, 10, -1], [-1, -2, 5]]\n");
    printf("b = [3, 15, 10]\n");
    printf("初始向量: x0 = [0, 0, 0]\n");
    printf("精度要求: %.0e\n", TOL);
    printf("\n");

    int iter_jacobi = jacobi(A, b, x_jacobi);
    printf("Jacobi迭代法:\n");
    printf("迭代次数: %d\n", iter_jacobi);
    print_vector(x_jacobi, "解为：");
    printf("\n");
    
    int iter_gs = gauss_seidel(A, b, x_gs);
    printf("Gauss-Seidel迭代法:\n");
    printf("迭代次数: %d\n", iter_gs);
    print_vector(x_gs, "解为");
    printf("\n");
    
    return 0;
}
```

# 4 收敛性分析

## 4.1 Jacobi迭代法的收敛性分析

步骤1：判断是否严格对角占优

对于系数矩阵 $A$：

$$A = \begin{bmatrix} 10 & -2 & -1 \\ -2 & 10 & -1 \\ -1 & -2 & 5 \end{bmatrix}$$

有：
- 第一行：$|a_{11}| = 10 > |a_{12}| + |a_{13}| = |-2| + |-1| = 3$
- 第二行：$|a_{22}| = 10 > |a_{21}| + |a_{23}| = |-2| + |-1| = 3$
- 第三行：$|a_{33}| = 5 > |a_{31}| + |a_{32}| = |-1| + |-2| = 3$

故矩阵 $A$ 是严格对角占优矩阵。



步骤2：Jacobi迭代矩阵的构造

Jacobi迭代法的迭代格式为：
$$x_i^{(k+1)} = \frac{1}{a_{ii}} \left( b_i - \sum_{j \neq i} a_{ij} x_j^{(k)} \right), \quad i=1,2,3$$

矩阵形式为：$\mathbf{x}^{(k+1)} = B_J \mathbf{x}^{(k)} + \mathbf{g}_J$
其中 $B_J = D^{-1}(L+U)$，$\mathbf{g}_J = D^{-1}b$，$D = \text{diag}(10, 10, 5)$，$L = \begin{bmatrix} 0 & 0 & 0 \\ -2 & 0 & 0 \\ -1 & -2 & 0 \end{bmatrix}$，$U = \begin{bmatrix} 0 & -2 & -1 \\ 0 & 0 & -1 \\ 0 & 0 & 0 \end{bmatrix}$。



步骤3：收敛性定理应用

由于A是严格对角占优的矩阵，故Jacobi迭代法对任意初始向量收敛



步骤4：谱半径计算与收敛速度估计

为了分析收敛速度，需要计算迭代矩阵 $B_J$ 的谱半径 $\rho(B_J)$。

由步骤2可得：

$$B_J = D^{-1}(L+U) = \begin{bmatrix}
0 & 0.2 & 0.1 \\
0.2 & 0 & 0.1 \\
0.2 & 0.4 & 0
\end{bmatrix}$$

求解特征方程 $\det(B_J - \lambda I) = 0$：

$$\begin{vmatrix}
-\lambda & 0.2 & 0.1 \\
0.2 & -\lambda & 0.1 \\
0.2 & 0.4 & -\lambda
\end{vmatrix} = -\lambda^3 + 0.10\lambda + 0.012 = 0$$

即 $\lambda^3 - 0.10\lambda - 0.012 = 0$。解此三次方程得到$B_J$三个实特征值：

$$\lambda_1 \approx 0.3646, \quad \lambda_2 \approx -0.199, \quad \lambda_3 \approx -0.165$$

所以谱半径为：
$$\rho(B_J) = \max_{1 \leq i \leq 3} |\lambda_i| = 0.3646$$

收敛速度分析：
迭代法的渐近收敛速度由谱半径决定，误差满足：
$$\|\mathbf{e}^{(k)}\| \sim C \cdot [\rho(B_J)]^k \|\mathbf{e}^{(0)}\|$$

其中 $\rho(B_J) = 0.3646$，即收敛因子 $L = 0.3646$。所以误差每迭代一次大约减少为原来的 $0.3646$ 倍。故要达到精度 $\epsilon = 10^{-5}$，理论上需要的迭代次数为：
$$k \approx \frac{\ln\epsilon}{\ln\rho(B_J)} = \frac{\ln(10^{-5})}{\ln(0.3646)} \approx \frac{-11.5129}{-1.0088} \approx 11.4 \text{次}$$



步骤5：实际验证

程序运行结果显示Jacobi迭代法经过13次迭代达到精度要求，与谱半径估计的理论迭代次数11.4次基本相符，收敛过程稳定，验证了理论分析的正确性。

## 4.2 Gauss-Seidel迭代法的收敛性分析

步骤1：Gauss-Seidel迭代矩阵的构造

Gauss-Seidel迭代法的迭代格式为：
$$x_i^{(k+1)} = \frac{1}{a_{ii}} \left( b_i - \sum_{j=1}^{i-1} a_{ij} x_j^{(k+1)} - \sum_{j=i+1}^{n} a_{ij} x_j^{(k)} \right)$$

写成矩阵形式：$\mathbf{x}^{(k+1)} = B_{GS} \mathbf{x}^{(k)} + \mathbf{f}_{GS}$
其中 $B_{GS} = (D-L)^{-1}U$（这里的$D，L，U$与Jacobi迭代法中一样。



步骤2：收敛性定理应用

由于A是严格对角占优的矩阵，故Gauss-Seidel迭代法收对任意初始向量收敛。



步骤3：谱半径计算与收敛速度估计

为了分析收敛速度，计算迭代矩阵 $B_{GS}$ 的谱半径。

由步骤1可得：

$$B_{GS} = (D-L)^{-1}U = \begin{bmatrix}
0 & 0.2 & 0.1 \\
0 & 0.04 & 0.12 \\
0 & 0.04 & 0.064
\end{bmatrix}$$

求解特征方程 $\det(B_{GS} - \lambda I) = 0$：

$$\begin{vmatrix}
-\lambda & 0.2 & 0.1 \\
0 & 0.04-\lambda & 0.12 \\
0 & 0.04 & 0.064-\lambda
\end{vmatrix} = -\lambda[(0.04-\lambda)(0.064-\lambda) - 0.12 \times 0.04] = 0$$

展开得：$-\lambda[\lambda^2 - 0.104\lambda - 0.00224] = 0$。

解得三个特征值：
$$\lambda_1 = 0, \quad \lambda_2 \approx 0.1223, \quad \lambda_3 \approx -0.0183$$

所以谱半径为：
$$\rho(B_{GS}) = \max\{|0|, |0.1223|, |-0.0183|\} = 0.1223$$

收敛速度分析：
渐近收敛速度由谱半径决定，误差满足：
$$\|\mathbf{e}^{(k)}\| \sim C \cdot [\rho(B_{GS})]^k \|\mathbf{e}^{(0)}\|$$

其中 $\rho(B_{GS}) = 0.1223$，收敛因子 $L_{GS} = 0.1223$。
误差每迭代一次大约减少为原来的 $0.1223$ 倍。

迭代次数估计：
要达到精度 $\epsilon = 10^{-5}$，理论上需要的迭代次数为：
$$k \approx \frac{\ln\epsilon}{\ln\rho(B_{GS})} = \frac{\ln(10^{-5})}{\ln(0.1223)} \approx \frac{-11.5129}{-2.1026} \approx 5.5 \text{次}$$

步骤5：实际验证

程序运行结果显示Gauss-Seidel迭代法仅需8次迭代就达到精度要求，与谱半径估计的理论迭代次数5.5次基本相符，且比Jacobi法的13次迭代少5次，收敛速度明显更快，验证了理论分析。

## 4.3 两种方法的收敛性对比

1. 收敛条件：两种方法都对严格对角占优矩阵收敛。
2. 收敛速度：Gauss-Seidel法通常比Jacobi法收敛更快。
3. 理论依据：
   - Jacobi法：谱半径 $\rho(B_J) = 0.3646$
   - Gauss-Seidel法：谱半径 $\rho(B_{GS}) = 0.1223$
4. 实际表现：
   - Jacobi法：13次迭代
   - Gauss-Seidel法：8次迭代
   - 收敛速度比：$\frac{13}{8} \approx 1.625$，Gauss-Seidel法快约62.5%

理论解释：Gauss-Seidel法在每次迭代中使用了最新计算出的分量值，信息利用更充分，因此收敛更快。

## 5.3 思考与总结

通过本实践题，我深刻理解了：

1. 迭代法的基本原理：Jacobi迭代法和Gauss-Seidel迭代法都是求解线性方程组的经典迭代方法，通过构造迭代格式逐步逼近精确解。

2. 收敛性条件的重要性：严格对角占优是保证迭代法收敛的充分条件。通过验证矩阵的严格对角占优性，可以预先判断迭代法的收敛性。

3. 收敛速度的差异：Gauss-Seidel法通常比Jacobi法收敛更快，这是因为Gauss-Seidel法在计算每个分量时都使用了最新计算出的值，信息利用更充分。但也需注意，也可能出现Jacobi法收敛但Gauss-Seidel法不收敛的情况，需对各方法的收敛性进行验证。

4. 理论与实际的结合：
   - 理论分析得到Jacobi法的收敛因子 $L_J = 0.3646$，Gauss-Seidel法的Sassenfeld数 $\beta = 0.1223$
   - 实际运行结果：Jacobi法13次迭代，Gauss-Seidel法8次迭代
   - 理论预测与实际结果相符，Gauss-Seidel法收敛更快

5. 数值稳定性：两种方法在合理初始值下都能稳定计算，未出现数值溢出或异常。

6. 方法选择策略：
   - 当矩阵严格对角占优时，两种方法都适用
   - 追求更快收敛速度时，优先选择Gauss-Seidel法
   - 需要并行计算时，Jacobi法更有优势（各分量可独立计算）
   - 对精度要求较高时，Gauss-Seidel法通常能提供更精确的解

7. 计算方法的应用价值：
   - 迭代法适用于大型稀疏线性方程组的求解
   - 通过理论分析可以预测算法的收敛性和收敛速度
   - 编程实现加深了对迭代法原理和数值计算过程的理解
